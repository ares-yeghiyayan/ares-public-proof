# Weekly Notes

This folder is for short weekly reviews of public proof work.

Weekly notes should connect daily progress into a clearer rhythm without becoming long reports.

## Purpose

A weekly note should show:

- what was shipped
- what improved
- what stayed consistent
- what should be tightened next
