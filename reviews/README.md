# Public Reviews

This folder is for short public-safe reviews of the public proof surface.

Reviews should make the repository easier to inspect without exposing private work.

## Purpose

A public review should check:

- whether the public route is clear
- whether boundary language is visible
- whether claims stay behind evidence
- whether notes remain short and reviewable
- whether private work remains private
